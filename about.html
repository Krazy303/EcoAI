Neural Architecture

Our classifier uses a **MobileNetV2** backbone—a streamlined deep neural network optimized for real-time mobile and web applications. By utilizing **Transfer Learning**, we've fine-tuned the model on thousands of waste-specific images to achieve high precision in distinguishing between subtle material differences.

  • 224x224 RGB Input
  • Depthwise Separable Convolutions
  • Softmax Output Layer
