# ♻️ EcoAI: Smart Waste Classifier

EcoAI is a modern, AI-powered web application designed to revolutionize waste management in India. By leveraging computer vision, EcoAI helps users instantly identify and sort waste into the correct disposal streams, promoting a circular economy.

## 🚀 Live Demo
Scan the **QR Code** in the "About" section or visit the deployed site to try the real-time classifier.

## ✨ Key Features
- **Smart Classifier**: Real-time AI classification using your webcam or image uploads.
- **Dual Camera Support**: Easily switch between selfie and rear cameras on mobile devices.
- **Impact Tracking**: Visualize India's waste stream distribution and your personal contribution.
- **Education Hub**: Learn "Pro Tips" for better recycling and composting habits.
- **Mobile First**: Fully responsive design with a native-app feel on smartphones.

## 🧠 Tech Stack
- **AI Engine**: TensorFlow.js with a fine-tuned **MobileNetV2** model.
- **Training**: Google Teachable Machine.
- **Frontend**: Tailwind CSS for a modern, "Eco-Professional" aesthetic.
- **Deployment**: Optimized for Vercel and GitHub Pages.

## 🛠️ Local Development

Due to browser security (CORS), you must use a local server to run the AI model.

### 1. Using Python (Quickest)
```bash
python3 -m http.server 8000
```
Then open [http://localhost:8000](http://localhost:8000)

### 2. Using Node/NPM
```bash
npm install
npm start
```
